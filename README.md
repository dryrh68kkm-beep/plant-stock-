## ✨ ฟีเจอร์

| | |
|---|---|
| 🌱 **เพิ่ม/แก้ไข/ลบ** | จัดการรายการต้นไม้ได้ครบ พร้อมรูปภาพ |
| 🏷️ **หมวดหมู่** | ไม้ประดับ, ไม้ฟอกอากาศ, ไม้อวบน้ำ, ไม้ดอก, ไม้ผล ฯลฯ |
| 🔍 **ค้นหา** | ค้นชื่อหรือหมวดหมู่ได้ทันที |
| 💰 **สรุปมูลค่าสต็อก** | จำนวนรายการ, จำนวนต้นรวม, มูลค่ารวมอัตโนมัติ |
| 📸 **อัปโหลดรูป** | ย่อขนาดอัตโนมัติ ไม่กินพื้นที่ |

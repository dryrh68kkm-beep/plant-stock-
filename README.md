## ✨ ฟีเจอร์

| | |
|---|---|
| 🌱 **เพิ่ม/แก้ไข/ลบ** | จัดการรายการต้นไม้ได้ครบ พร้อมรูปภาพ |
| 🏷️ **หมวดหมู่** | ไม้ประดับ, ไม้ฟอกอากาศ, ไม้อวบน้ำ, ไม้ดอก, ไม้ผล, **ไม้มงคล** ฯลฯ |
| 💾 **บันทึกข้อมูลถาวร** | เชื่อมกับฐานข้อมูลกลาง (Firebase Firestore) — ทีมทุกคนเห็นสต็อกอัปเดตพร้อมกันแบบเรียลไทม์ 🔥 ถ้าเชื่อมต่อไม่ได้จะสลับไปใช้ localStorage ของเครื่องแทนอัตโนมัติ |
| 🔍 **ค้นหา** | ค้นชื่อหรือหมวดหมู่ได้ทันที |
| 💰 **สรุปมูลค่าสต็อก** | จำนวนรายการ, จำนวนต้นรวม, **ต้นทุนรวม**, มูลค่ารวม คำนวณอัตโนมัติ |
| 📋 **หน้า 2 · ตารางสต็อก** | สลับดูข้อมูลทั้งหมดเป็นตารางแถวเดียว (รูป, ชื่อ, หมวดหมู่, ราคา, ต้นทุน, จำนวน, มูลค่ารวม) เหมาะกับดูภาพรวมหรือเช็คสต็อกเร็วๆ |
| 📸 **อัปโหลดรูป** | ย่อขนาดอัตโนมัติ ไม่กินพื้นที่ |
| 📖 **ฐานข้อมูลพันธุ์ไม้ในตัว** | พิมพ์ชื่อต้นไม้ยอดนิยม (กว่า 80 ชนิด เช่น มะลิ, มอนสเตอร่า, เฟื่องฟ้า, มะม่วง, กะเพรา ฯลฯ) แล้วกดปุ่ม ระบบเติมหมวดหมู่และคำแนะนำการดูแลให้ทันที — ใช้งานได้แบบออฟไลน์ ไม่ต้องต่อเน็ต |
| 🍀 **ความหมายมงคล** | ต้นไม้มงคลจะมีริบบิ้น 🍀 และข้อความความเชื่อแสดงเด่นบนการ์ดสินค้าโดยตรง |
